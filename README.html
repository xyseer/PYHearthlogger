功能介绍

  1. 自动登录
    检测到没有正在运行的游戏&兄弟的话会进入自动登录阶段, 会自动上号(暴雪要设置为不自动登录, 也就是把密码框下面的勾去掉), 游戏开启后会自动开启兄弟并启动挂机
  2. 掉线重连
    通过检测兄弟的胜负统计和当前所有窗口是不是正常来判断游戏是否已掉线(默认的设置为1200s检测一次, 一般这段时间里偶数萨, 奥秘法都会至少完成一把, 如果正在用互投模式可以把config.json中的"TIME_PER_GAME"值改小一点, 这样发生掉线后会更快的检测到以进入自动重连阶段)

配置介绍

config.json文件的解释

  • 该文件用于记录本软件运行中需要知道的一些信息

路径类

  • BUDDY_ROOT_PATH兄弟的根目录, 也就是C:\Users\lxr\Desktop\炉石兄弟汉化版2020.1.17所在的文件夹的路径.
  • CLIENT_PATH暴雪战网客户端的路径, 在暴雪战网的快捷方式上右键-打开文件的所在位置就可以找到

窗口类

"CLIENT_LOGIN_WINDOW_NAME": "暴雪战网登录",
"CLIENT_WINDOW_NAME": "暴雪战网",
"GAME_WINDOW_NAME": "炉石传说",
"BUDDY_WINDOW_NAME": "炉石兄弟博客【studyhb.cn】联合【炉石兄弟吧】修复汉化，禁止用于商业",
  • 从上到下的几个NAME大家自行理解一下, 这些窗口的名字用于寻找窗口的句柄, 一般如果你的暴雪客户端语言是简体中文的话这些字段都不用改, 如果已经改成了英文的话再改回到中文或把这几个字段的内容改成相应的英文名称即可

账号类

"ACCOUNTS": [{
    "ACCOUNT": "xxx@163.com",
    "PASSWORD": "ddd"
}, {
    "ACCOUNT": "ddd@163.com",
    "PASSWORD": "ddd"
}],
"ACTIVE_ACCOUNT": 2,
  • ACCOUNTS: 记录了当前挂机登录时的账号, 可以有多个
  • ACTIVE_ACCOUNT: 记录了当前使用的账号. 要挂第一个账号就把这地方改成1, 挂第二个就把这改成2

延迟类

 "NORMAL_DELAY": 1,
 "CLIENT_DELAY": 8,
 "LOGIN_DELAY": 10,
 "BUDDY_START_DELAY": 45,
 "GAME_START_DELAY": 10,
 "TIME_PER_GAME": 1200,
  • 启动各个软件需要等待的时间, 均以s为单位, 依次是:
    1. 执行普通操作的延迟, 一般改成1,2左右即可
    2. 启动暴雪战网的延迟, 10s左右即可
    3. 从点击登录按钮到出现暴雪战网窗口的时间, 一般在10s左右
    4. 兄弟启动的时间的上限, 这个因电脑配置不同差异巨大, 保险起见可以改到150s左右
    5. 游戏启动的时间, 也就是从点击"进入游戏"到游戏启动完成的时间, 10s左右即可
    6. 每一局对战花费的时间(估计), 也就是检测当前实例是否都在正常运行的周期, 1200s我觉得很足够了

运行方式

运行源码

  • 推荐运行源码, 这样运行中出现了问题可以有明显的报错, 方便向我汇报
  • 本工具是个小项目, 也就几百行代码, 而且环境比较简单, 还算好配置

安装Python 3.7.7

  • 点此链接下载: https://www.python.org/ftp/python/3.7.7/python-3.7.7-amd64.exe
  • 安装

    注意一定要勾选Add Python to PATH, 之后点击install now即可
  • 检查是否已正确安装
    • 打开cmd窗口, 输入python, 为图示就没问题了

安装vscode

  • 点此链接下载: https://vscode.cdn.azure.cn/stable/0ba0ca52957102ca3527cf479571617f0de6ed50/VSCodeUserSetup-x64-1.43.2.exe
  • 安装过程略
  • 选中relogger的文件夹, 右键以vscode打开

安装支持的库

  • 分别在cmd中运行以下的命令
    pip install pywin32
pip install mss

将Python和vscode的默认运行方式均改为以管理员权限运行

  • python目录为: C:\Users\你的账号名\AppData\Local\Programs\Python\Python37
  • vscode目录为: C:\Users\lxr\AppData\Local\Programs\Microsoft VS Code

vscode打开项目后进入main.py, 按下f5运行即可
